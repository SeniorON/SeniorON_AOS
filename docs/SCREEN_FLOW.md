# 시니어 ON 화면 목록 및 플로우

이 문서는 시니어 ON Android 앱의 화면 목록과 화면 간 이동 흐름을 정리합니다.

## 전체 구조

```text
앱 실행
-> 스플래시 화면
-> 모드 선택 화면
-> 로그인 화면 또는 회원가입 시작 화면
-> 가족공유코드 확인 화면
-> 자녀 메인 화면 또는 부모님 런처화면
```

## 인증 플로우

### 로그인

```text
로그인 화면
-> 아이디 입력
-> 비밀번호 입력
-> 로그인 성공
-> 가족공유코드 확인 화면
```

로그인 화면에서 이동 가능한 기능:

- 회원가입
- 아이디 찾기
- 비밀번호 찾기
- 카카오 로그인
- 구글 로그인

### 회원가입

```text
회원가입 시작 화면
-> 회원가입 정보 입력 화면
-> 이메일 인증 화면
-> 계정 정보 입력 화면
-> 약관 동의 화면
-> 가족공유코드 확인 화면
```

회원가입 단계:

1. 가입 방식 선택
2. 이름과 생년월일 입력
3. 이메일 인증
4. 아이디 중복 확인
5. 비밀번호 입력과 확인
6. 약관 동의
7. 가족공유코드 확인

### 아이디 찾기

```text
아이디 찾기 화면
-> 이름 입력
-> 이메일 입력
-> 아이디 찾기
-> 결과 표시
```

### 비밀번호 찾기

```text
비밀번호 찾기 화면
-> 이름 입력
-> 아이디 입력
-> 이메일 인증번호 전송
-> 인증번호 확인
-> 비밀번호 재설정 화면
-> 비밀번호 변경 완료 화면
```

## 가족공유코드 플로우

```text
가족공유코드 확인 화면
-> 가족공유코드 있음
   -> 가족공유코드 입력 화면
   -> 메인 화면

-> 가족공유코드 없음
   -> 가족공유코드 생성 화면
   -> 부모님 정보 입력 화면
   -> 메인 화면
```

가족공유코드는 구성원, 건강 정보, 알림, 사진 공유를 연결하는 기준입니다.

## 자녀 모드 플로우

```text
자녀 메인 바텀네비 구조
-> 화면 탭(화면 지원)
-> 건강 탭
-> 알림 탭
-> 가족 탭
-> 설정 탭
```

## 화면 탭(화면 지원)

```text
화면 탭(화면 지원)
-> 부모님 폰 미리보기
-> 버튼 편집 화면
   -> 버튼 이름 편집
   -> 버튼 아이콘 설정
   -> 버튼 삭제
   -> 버튼 순서 변경
   -> 새 버튼 추가
-> 글씨 편집 화면
   -> 크게 / 보통 / 작게 선택
-> 부모님 런처화면 반영
```

주요 기능:

- 부모님 기기 연결 상태 확인
- 배터리 상태 확인
- 부모님 런처화면 미리보기
- 최대 8개 버튼 관리
- 부모님 런처화면 글씨 크기 조절

## 건강 탭

```text
건강 탭
-> 복약 관리 화면
   -> 약 추가 화면
-> 병원 일정 화면
   -> 진료 추가 화면
-> 진료비 공동 관리 화면
```

주요 기능:

- 복약 현황 확인
- 약 추가와 반복 요일 설정
- 병원 일정 캘린더 확인
- 진료 등록, 수정, 삭제
- 진료비 공동 관리

## 알림 탭

```text
알림 탭
-> SOS 알림 상세 화면
-> 무활동 감지 상세 화면
-> 위험 링크 감지 상세 화면
-> 외출/귀가 알림 상세 화면
-> 무활동 감지 시간 설정 화면
```

주요 기능:

- SOS 알림
- 무활동 감지
- 위험 링크 감지
- 외출/귀가 알림
- 상세 화면에서 전화 걸기, 길찾기, 닫기

알림 기능은 부모님 휴대폰의 권한 허용을 전제로 설계합니다.

## 가족 탭

```text
가족 탭
-> 가족 구성원 목록
-> 가족 추가
-> 구성원 설정 화면
   -> 주 담당자 변경
   -> 구성원 삭제
-> 가족 사진 공유 화면
   -> 사진 더보기
   -> 사진 업로드
      -> 갤러리에서 선택
      -> 카메라 촬영
```

가족 탭은 가족공유코드로 연결된 구성원들이 함께 사용하는 공간입니다.

## 설정 탭

```text
설정 탭
-> 내 계정
   -> 자녀 프로필 설정 화면
   -> 비밀번호 변경 화면
-> 연결된 기기
-> 멤버십
-> 도움말·문의
-> 의견 보내기
-> 로그아웃
```

## 부모님 런처화면

```text
부모님 런처화면
-> 날짜 / 시간 / 날씨 확인
-> 음악 카드
-> 일정 카드
-> 큰 버튼 그리드
   -> 전화
   -> 메시지
   -> 말벗
   -> 복약 또는 건강
   -> 카메라
   -> 사진
   -> 일정
   -> 긴급알림
```

부모님 런처화면은 보호자 휴대폰의 화면 탭(화면 지원)에서 버튼 배치와 글씨 크기를 수정할 수 있습니다.

## 화면 ID 참고

문서 본문에서는 화면 이름을 한글로 작성하고, 구현 파일과 연결할 때만 스크린 ID를 사용합니다.

| 화면 이름 | 스크린 ID |
| --- | --- |
| 모드 선택 화면 | ModeSelectionScreen |
| 로그인 화면 | LoginScreen |
| 회원가입 시작 화면 | SignUpStartScreen |
| 가족공유코드 확인 화면 | FamilyShareCodeCheckScreen |
| 자녀 메인 바텀네비 구조 | ChildMainScreen |
| 화면 탭(화면 지원) | DisplayTabScreen |
| 건강 탭 | HealthTabScreen |
| 알림 탭 | NotificationTabScreen |
| 가족 탭 | FamilyTabScreen |
| 설정 탭 | SettingsTabScreen |
| 부모님 런처화면 | ParentLauncherScreen |
